# saiden-dev/homebrew-tap

Custom Homebrew formulas for Saiden and related projects.

## Installation

```bash
brew tap saiden-dev/tap
```

## Formulas

| Formula | Description | Install |
|---------|-------------|---------|
| **[hu](https://github.com/saiden-dev/hu)** | Token-efficient dev workflow CLI for Claude Code | `brew install saiden-dev/tap/hu` |
| **[browse](https://github.com/saiden-dev/browse)** | Headless browser automation for Claude Code | `brew install saiden-dev/tap/browse` |
| **[icloud-status](https://github.com/saiden-dev/icloud-status)** | macOS CLI for iCloud status information | `brew install saiden-dev/tap/icloud-status` |
| **[tengu-init](https://github.com/tengu-apps/tengu-init)** | Provision Tengu PaaS servers | `brew install saiden-dev/tap/tengu-init` |
| **[thumbsdown](https://github.com/aladac/thumbsdown)** | Generate thumbnail grids from video files | `brew install saiden-dev/tap/thumbsdown` |

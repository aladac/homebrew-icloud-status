# Homebrew Tap

Custom Homebrew formulas.

## Installation

```bash
brew tap saiden-dev/tap
```

## Formulas

### icloud-status

macOS CLI tool to display comprehensive iCloud status information.

```bash
brew install saiden-dev/tap/icloud-status
```

Or:

```bash
brew tap saiden-dev/tap
brew install icloud-status
```

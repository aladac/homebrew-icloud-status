# saiden-dev/homebrew-tap

Custom Homebrew formulas for Saiden and related projects.

## Installation

```bash
brew tap saiden-dev/tap
```

## Formulas

| Formula | Description | Install |
|---------|-------------|---------|
| **[hu](https://github.com/saiden-dev/hu)** | Token-efficient dev workflow CLI for Claude Code | `brew install hu` |
| **[browse](https://github.com/saiden-dev/browse)** | Headless browser automation for Claude Code | `brew install browse` |
| **[icloud-status](https://github.com/saiden-dev/icloud-status)** | macOS CLI for iCloud status information | `brew install icloud-status` |
| **[tengu-init](https://github.com/tengu-apps/tengu-init)** | Provision Tengu PaaS servers | `brew install tengu-init` |
| **[thumbsdown](https://github.com/aladac/thumbsdown)** | Generate thumbnail grids from video files | `brew install thumbsdown` |

> After tapping, all formulas are available as `brew install <name>` without the full path.
